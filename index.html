FAQ

Do I need prior coding experience?

Helpful, but not required. We begin from fundamentals and ramp up with weekly practice. If you're brand‑new, we’ll share a short prep pack (syntax basics + problem‑solving patterns) before Week‑1.

  • Optional pre‑work: variables, loops, functions, arrays
  • Bridge sessions for absolute beginners during Week‑1
  • Office hours for catch‑up questions
Which languages do you use?

We use C to build low‑level clarity (pointers, memory), Python for algorithms and quick iteration, and Java for OOP perspective. The goal is thinking skills that transfer across languages.

How much time per week?

~6–8 hours: 2 live sessions + assignments + optional labs. Expect steady, manageable work rather than cramming.

  • Live: 2 × 90‑min
  • Practice: 2–3 hrs problem sets
  • Capstone weeks: add 1–2 hrs
Is there 1:1 help?

Yes. Instructor office hours, async feedback on code, and quick DM help in our Discord. We also run short clinics on tough topics (recursion, DP, pointers).

Will sessions be recorded?

Yes, recordings are shared within 24 hours and available through the cohort window. We also provide annotated notes and solution walkthroughs.

What do I need installed?

A laptop with internet access. We’ll guide you to install VS Code, a C compiler (gcc / clang), Python 3, and Git. Setup help is included in Week‑0.

Do you offer scholarships or refunds?

Yes — limited merit/need‑based scholarships. If the program isn’t a fit after Week‑1, we offer a full refund. Write to hello@byteboost.site.

How are students assessed?

Through weekly problem sets, a light mid‑module check, and the final capstone. Rubrics emphasize clarity, correctness, and communication.
